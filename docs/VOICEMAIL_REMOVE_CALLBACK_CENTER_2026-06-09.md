# Voicemail Update: Callback Center Removed (2026-06-09)

The voicemail page no longer renders the Callback Center section.

## Scope
- Removed callback-center dialer and contact-search section from the voicemail UI.
- Removed callback-center-only state and helper code from the voicemail page.
- Removed row-level Call back button that fed into the removed section.
